Electronics and IoT class project, Spring 2025

IoT Speaker Remote

With Truman O'Brien ESP32 Wi-Fi protocol Low-power design

A custom wireless remote and embedded speaker-side controller built to replace the unavailable infrared remote for an Audioengine A5+ speaker set.

Project overview

Our solution replaced the original IR interface with an ESP32-based IoT system that enables real-time, bidirectional communication between a 3D-printed handheld remote and a custom controller embedded within the speaker.

The remote features a rotary encoder for volume and mute control, a low-power alphanumeric display, and a sleep mode that substantially reduces power draw during inactivity.

Speaker-side controller

Intercepts and emulates encoder signals to adjust volume through GPIO while preserving compatibility with the original analog control circuit.

Handheld remote

Uses a rotary encoder, compact display, and low-power firmware to create a durable and modern replacement for the original remote.

Wireless communication

Handles communication over Wi-Fi using a custom message protocol with input queuing for reliability and responsive volume changes.

Engineering challenges

  • Reverse-engineering the speaker’s internal control logic.
  • Implementing robust low-power firmware for long recharge intervals.
  • Synchronizing physical and digital volume states.
  • Packaging the electronics in a clean, ergonomic enclosure.
