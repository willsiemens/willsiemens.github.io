IoT Remote

Mechatronics & Embedded Systems

Designed and built a custom IoT-enabled remote to bridge the gap between physical user interfaces and smart-home connectivity. This project required integrating custom PCB housing with ergonomic mechanical design.

System Integration

Integrated wireless modules with low-power microcontrollers, ensuring responsive latency and long battery life for a seamless user experience.

Prototyping

The external casing was designed in SolidWorks with a focus on ergonomics and 3D-printed using high-resolution resin to achieve a professional finish.

Core Technologies

  • Hardware: ESP32/Arduino, Li-Po Charging Circuits.
  • Software: C++, MQTT Protocol for IoT communication.
  • Design: Rapid Prototyping and DFM (Design for Manufacturing).
